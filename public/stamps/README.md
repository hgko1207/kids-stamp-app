# 도장 이미지 폴더

이 폴더에 캐릭터 이미지 파일을 넣으세요.

## 지원 형식
- PNG (추천 — 배경 투명 처리된 것)
- JPG
- WebP
- GIF

## 권장 파일명 (예시)
```
carbot1.png   ← 헬로카봇 알타
carbot2.png   ← 헬로카봇 레온
carbot3.png   ← 헬로카봇 하이파이
carbot4.png   ← 헬로카봇 팡팡이
...
```

## 앱에서 사용하는 방법

파일을 이 폴더에 넣은 후, 앱의 **부모 설정 → 도장 이미지 관리 → + 버튼**에서
아래 형식으로 입력:

```
/stamps/carbot1.png
/stamps/carbot2.png
```

## 이미지 크기 권장
- 최소 100×100px
- 정사각형 비율 (1:1)
- PNG 투명 배경이면 더 예쁘게 보여요
